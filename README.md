## Node JS Starter

### Frameworks to use

* [Express](https://expressjs.com/)
* [Koa](https://koajs.com/)
* [Fastify](https://www.fastify.io/)

### Node
* [Volta](https://volta.sh/)
* [NPM](https://www.npmjs.com/)


### Getting Started
```
curl https://get.volta.sh | bash

volta install node
volta install yarn

git init
yarn init

yarn add fastify --exact
yarn add nodemon --exact --dev
```

### Installing
* `yarn install`

### Running
* `node server.js`
* `npx nodemon server.js`
